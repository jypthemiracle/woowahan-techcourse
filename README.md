# java-chess
체스 게임 구현을 위한 저장소

1. ChessGame
    + 체스 판 초기화
    + 말 배치
    + 현재 차례인 플레이어 저장
2. ChessBoard
    + 말들 저장
    + 말의 위치 변경
3. Piece
    + 이동 규칙 저장
    + 이동 규칙에 따른 경로 반환
    + 현재 위치 저장
    + 플레이어 저장
    + 점수 저장 
4. Path
    + 좌표 A부터 B까지의 경로 저장
5. Coordinate
    + 좌표 값 저장
6. Position
    + x,y 좌표 저장 
7. Player
    + 플레이어 차례 저장
8. PlayerScore
    + 플레이어의 점수 저장
